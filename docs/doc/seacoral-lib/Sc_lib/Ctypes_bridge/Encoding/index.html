val pp_c_ptr_decode : (string * string * string * string) Basics.PPrt.pp

pp_c_ptr_decode fmt (v, d) prints using fmt a C expression that computes a length of array encoded by a given unsigned long variable v, at a depth encoded by an unsigned int d.

val pp_cstring_buff_length_decode : (string * string) Basics.PPrt.pp

pp_cstring_buff_length_decode fmt (v, d) prints using fmt a C expression that computes a length of C character string, including the trailing '\000', encoded by a given unsigned long variable v, at a depth encoded by an unsigned int d.

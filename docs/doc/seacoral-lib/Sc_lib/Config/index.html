val as_module : salt:string -> tag:string -> (module Core) -> config -> (module T) Lwt.t

as_module ~salt ~tag config returns a module using the configuration value in argument and output directory properly setup if successful, an error message otherwise.

salt is a string identifier that is added to the project hash, calculated from the different tools options set in the configuration file.

In turn, tag is a user-readable string for naming project directories and/or symlinks: it should be constructed based on project options like coverage criteria.

val get_config : unit -> config

get_config () Returns the configuration of the project from the command line arguments and the configuration file, if provided. Options set on the configuration file override the command line options.

Logs reporting

val reporter : ?max_level:Logs.level -> ?time_precision:int -> Stdlib.Format.formatter -> Logs.reporter * Logs.level option Stdlib.ref

reporter ?max_level ?time_precision ppf returns both a properly setup Logs reporter, and a reference to the level below which the reporter does not report log messages.

The returned referenced level !max_level is used to filter out messages: if it is None, nothing is printed; if it is Some l, then every message whose level is above l is printed.

time_precision is used to enable and control a timestamp prefix: if it is 0 (the default) or negative, then no timesamp is prepended. Otherwise if it is p, then each message is prepended with the number of seconds elapsed since the tool process started, measured with precision p and put in bewteen square brackets.

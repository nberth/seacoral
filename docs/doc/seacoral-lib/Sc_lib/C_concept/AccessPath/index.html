Module C_concept.AccessPath

Access paths for aggregated C values (structs, unions, arrays)

type _ t

The representation of a C value. It can be atomic (atomic t) or aggregating values (normal/reversed t). The normal and reversed represent the same value, but the reversed version makes the append functions slightly more efficient.

val append : 'k t -> atomic t -> reversed t

Extends t with the atomic element a.

val append_f : 'k t -> ?deref:bool -> string -> reversed t

append_f t ?deref str

Extends t with a field str. If deref is set to false (false by default), it represents a simple data; otherwise, it represents a pointer to the data.

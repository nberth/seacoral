Printers

val pp_lst : ?fopen:ufmt -> ?fsep:ufmt -> ?fclose:ufmt -> ?fempty:ufmt -> 'a pp -> 'a list pp

pp_lst ~fopen ~fsep ~fclose ~fempty pp_e Returns a list printer with a custom open delimiter (fopen), a custom close delimiter (fclose) and a custom separator (fsep). If the list is empty, prints (fempty).

val pp_lst_i : ?fopen:ufmt -> ?fsep:ufmt -> ?fclose:ufmt -> ?fempty:ufmt -> ('a * int) pp -> 'a list pp

Same as pl, but with a counter incremented for each element of the list.

val pp_lstset : ?fsep:ufmt -> 'a pp -> 'a list pp

Same as pl, but prints the list with a set-like notation (lt1, elt2, ...)

val pp_arr : ?fopen:ufmt -> ?fsep:ufmt -> ?fclose:ufmt -> ?fempty:ufmt -> 'a pp -> 'a array pp

Same as pl, but for arrays.

val pp_arr_i : ?fopen:ufmt -> ?fsep:ufmt -> ?fclose:ufmt -> ?fempty:ufmt -> ('a * int) pp -> 'a array pp

Same as pli, but for arrays.

val pp_nodelim : ?fsep:ufmt -> ?fterm:ufmt -> 'a pp -> 'a list pp

Same as pl, but without an opening delimiter.

val pp_nodelim_i : ?fsep:ufmt -> ?fterm:ufmt -> ('a * int) pp -> 'a list pp

Same as pp_no_delim, but with a counter.
